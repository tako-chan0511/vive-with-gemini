## 1. 表紙：コンセプト提示

![P１：Toppage技術と遊び心のデジタルプレイグラウンド](/images/gamehub/P１：Toppage技術と遊び心のデジタルプレイグラウンド.jpg)

* **狙い**：Topgageの全体像（「技術 × 遊び心」）を一目で理解させる導入。

  * Nuxt3 / Vue3 / AI を軸にした「多機能Webアプリのハブ」。

## 2. Topgageとは？

![P２：Toppageとは？](/images/gamehub/P２：Toppageとは？.jpg)

* **狙い**：単なるポートフォリオではなく、**アプリ集合体**である点を明確化。
* **伝えたいこと**

  * VercelホスティングのWebアプリが**25+** 集約。
  * **登録不要**でブラウザから即体験可能。
  * **3カテゴリ（Game / AI / Utility）** で整理。
  * モダンスタック（Nuxt3 / Vue3 / Next.js / Vercel）。
* **ポイント**

  * 「Topgageは“作品置き場”ではなく、25以上のWebアプリをまとめた体験型ハブです。登録不要で、その場で試せます。」

## 3. 3つの柱

![P３：３つの柱で構成される体験](/images/gamehub/P３：３つの柱で構成される体験.jpg)

* **狙い**：提供価値を“3本柱”で整理し、理解を早める。

  2. **AI & Innovation（革新）**：Gemini等で新しい対話/体験

* **ポイント**

  * 「Topgageは“遊び”“AI”“実用”の3軸でアプリを展開し、楽しさと学びを両立させています。」
